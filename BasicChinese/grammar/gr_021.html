Chinese Grammar - 把 - construction

Index
The 把 construction is specific to Chinese. 把 is inserted before the object and the construction preceedes the verb. The normal sentence order SVO then becomes S把OV.

学生把作业做完了。 xuésheng bǎ zuòyè zuò wán le The student finished the homework.


The 把 construction can only be used with:
  1. transitive verbs
  2. the objective must be definite (not some abstract concept)
  3. the verb must modify the object in some way
  4. the verb must be followed by some other element (e.g. a complement, 了)
